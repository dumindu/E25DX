## Add Main Menu Items

1. Add new `menus` block to `hugo.yaml` to add menu items

   ```yaml
   menus:
     main:
       - identifier: home
         name: Home
         pageRef: /
         weight: 1
         params:
           icon: home
       - identifier: docs
         name: Documentation
         pageRef: /docs
         weight: 2
         params:
           icon: menu_book
       - identifier: blog
         name: Blog
         pageRef: /blog
         weight: 3
         params:
           icon: news
       - identifier: about
         name: About Me
         pageRef: /page/about
         weight: 4
         params:
           icon: account_circle
   ```

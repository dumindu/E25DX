## Add Main Menu Items

1. Add new `menus` block to `hugo.yaml` to add menu items.

   ```yaml
   menus:
     main:
       - identifier: home
         name: Home
         pageRef: /
         weight: 1
         params:
           icon: home
       - identifier: docs
         name: Documentation
         pageRef: /docs
         weight: 2
         params:
           icon: menu_book
       - identifier: blog
         name: Blog
         pageRef: /blog
         weight: 3
         params:
           icon: news
       - identifier: about
         name: About Me
         pageRef: /page/about
         weight: 4
         params:
           icon: account_circle
   ```

2. Update/ add new icons.

   ```shell {title="Folder Structure"}
   ├── themes
   │   └── E25DX
   │      └── assets
   │         └── icons
   │
   └── assets
       └── icons
           └── account_circle.svg
   ```
   - Create a new folder `assets/icons` to save new SVG files.
   - Currently, we use [Google's Material Symbols](https://fonts.google.com/icons); Optical Size-"20px" and Style-"Material symbols", "Rounded"
   - Make sure set `height="20px" width="20px"` and remove `fill=""` on SVG to support theme colors.

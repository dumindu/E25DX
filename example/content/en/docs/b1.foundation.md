## Install Hugo

- Use prebuilt binaries via https://github.com/gohugoio/hugo/releases/latest
- Verify the installation via `hugo version`

> [!note]
> Refer https://gohugo.io/installation for troubleshooting.

## Install Pagefind

- Use prebuilt binaries via https://github.com/pagefind/pagefind/releases
- Verify the installation via `pagefind --version`

> [!note]
> Refer https://pagefind.app/docs/installation for troubleshooting.

---

## Useful Commands

- `hugo serve` : Run Hugo site locally.
- `hugo build --minify` : Build Hugo site for production.
- `pagefind --site public` : Generate pagefind indexes for the Hugo generated site.

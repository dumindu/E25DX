## Create Separate Site Pages

1. Add `content/en/page/about/index.md` for about page.

   ```markdown
   ---
   title: About
   url: "about"
   ---

   About/ Any Markdown
   ```

2. Add any image of png (or jpg) for `content/en/page/about/cover.png` for about page cover image.
